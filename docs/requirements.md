# 短视频泰语学习播放器 — 需求文档

## 项目背景

用户通过短视频学习泰语，需要反复听每个句子来加深记忆。现有播放器无法按句子粒度重复播放，也无法自动生成双语字幕辅助理解。

## 目标用户

泰语学习者，使用 2015 款 Mac（无 GPU），视频来源为下载的 MP4 短视频。

## 需求列表

### P0 — 核心需求：自动断句 + 按句重复播放

| 编号 | 功能 | 描述 |
|------|------|------|
| F1 | 导入视频 | 支持导入本地 MP4 文件 |
| F2 | 自动断句 | 调用在线语音识别 API，自动将音频按句子/短语切分，返回每句的起止时间和原文文本 |
| F3 | 按句重复播放 | 播放到句尾时自动回跳到句首重复播放，用户可设定每句重复遍数（如 3/5/10 遍） |
| F4 | 句子列表面板 | 在视频旁显示所有句子列表，当前播放句子高亮，点击任意句子可跳转 |
| F5 | 播放控制 | 上一句/下一句按钮、暂停/继续、调整播放速度（0.5x-2x） |

### P1 — 次要需求：双语字幕

| 编号 | 功能 | 描述 |
|------|------|------|
| F6 | 自动翻译 | 调用 DeepSeek API 将泰语/英语原文翻译为中文 |
| F7 | 双语字幕显示 | 视频上叠加显示双语字幕，上行原文、下行中文 |
| F8 | 字幕开关 | 可独立控制显示/隐藏原文字幕和中文字幕 |

### P2 — 最低优先级：断句编辑

| 编号 | 功能 | 描述 |
|------|------|------|
| F9 | 编辑断句 | 当自动断句不准确时，允许手动调整句子的起止时间 |
| F10 | 合并/拆分句子 | 支持将相邻句子合并或拆分一个句子为两个 |

## 非功能需求

- 视频格式：MP4（HTML5 原生支持）
- 运行环境：2015 款 Mac，无 GPU，所有 AI 处理走在线 API
- 语言识别：支持泰语（th）和英语（en）
